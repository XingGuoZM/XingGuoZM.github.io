# webpack工作原理

webpack是一个js模块打包工具，通过js模块之间导入导出的依赖关系，构建依赖网络、静态分析来打包合并。其基本原理：
```
1. 获取参数，根据参数初始化编译器（compiler）
2. 根据entry配置找到入口文件
3. 编译，调用所有配置的loader对文件进行转化，递归，最终得到处理后的内容和他们的依赖关系
4. 根据配置输出路径和文件名，将内容写入文件
```
